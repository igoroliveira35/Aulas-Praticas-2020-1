Lista de Definições

Linguagem de Marcação
É a linguagem utilizada para criar a estrutura de uma página web.
HTML é a linguagem de marcação mais popular.
Linguagem de Formatação
É a linguagem utilizada para formatar os elementos de uma página web.
CSS é a linguagem de formatação mais popular.
Linguagem de Programação
É a linguagem utilizada para inserir lógica de programação em uma página web.
JavaScript é a linguagem de programação mais popular.
